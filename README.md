# Deep Learning

This repository contains a collection of homework assignments completed as part of a deep learning course during my M.Sc. in Computer Science at the University of Haifa.

## 📚 Repository Structure

The repository is organized by assignment, with each folder containing the relevant Jupyter Notebooks, source code, and reports:

* **hw1**: Logistic Regression, Feedforward Neural Networks, Backpropagation, Optimization, Batch Normalization.
* **hw2**: Regularization, Batch Normalization, Convolutional Neural Networks, Semantic Segmentation.
* **hw3**: RNNs, GRUs, LSTMs, Transformers.
* **hw4**: Word Embeddings, Transformers, Vision Transformers, Few-Shot Learning, Self-Supervised Learning.


## 🛠️ Technologies
* **Language:** Python 
* **Framework:** PyTorch
* **Environment:** Jupyter Notebook / Google Colab
